INTRODUCTION				*ddc-source-slash-commands-introduction*

*ddc-source-slash-commands* is a ddc.vim source that provides completion for
Claude Code slash commands (also known as "custom prompts" in Codex CLI).

When you type "/" in your buffer, it suggests available slash commands from
your commands directory.


==============================================================================
INSTALL						*ddc-source-slash-commands-install*

This plugin requires the following:

- Vim/Neovim with denops.vim
- ddc.vim


==============================================================================
EXAMPLES					*ddc-source-slash-commands-examples*

Basic configuration:
>
	call ddc#custom#patch_global('sources', ['slash_commands'])

	call ddc#custom#patch_global('sourceOptions', {
	  \ 'slash_commands': {
	  \   'mark': '[cmd]',
	  \   'matchers': ['matcher_head'],
	  \   'minAutoCompleteLength': 1,
	  \ }})
<

With custom parameters:
>
	call ddc#custom#patch_global('sourceParams', {
	  \ 'slash_commands': {
	  \   'commandsDir': '~/my/commands/dir',
	  \   'extensions': ['.md', '.txt'],
	  \ }})
<


==============================================================================
PARAMS						*ddc-source-slash-commands-params*

					*ddc-source-slash-commands-param-commandsDir*
commandsDir	(string)
		Directory path containing slash command files.

		Default: "~/.config/claude/commands"

					*ddc-source-slash-commands-param-extensions*
extensions	(string[])
		File extensions to include in completion.

		Default: [".md"]
